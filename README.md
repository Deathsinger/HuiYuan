会员项目

需求分析
伴随着信息行业的蓬勃发展和人们办公自动化意识的增强，会员管理部门的工作也越来越繁重，原来的会员管理系统已经不能完全满足相关人员使用的需要。为了协助信息行业开展会员管理工作，提高工作效率，充分利用信息行业的现有资源，开发更好的会员管理系统势在必行。可以做到真正实现节约资源、提高效率、业务处理的同时实现会员管理系统的功能作用。
本次开发会员管理系统采用B/S（浏览器/服务器）结构，管理员和会员用户有不同的使用权限。主要内容如下：
对于管理员所具有权限的使用模块
1.	登录管理:系统的注册登录以及修改密码；
2.	会员管理：
a)	等级管理：根据会员积分情况，为会员设置等级；
b)	会员信息管理：对会员个人信息的增删改查,以及会员积分的查询；
c)	会员卡解锁和绑定
3.	兑换商品：实现会员用户使用积分可以兑换相应的商品；
4.	商品管理：管理员对商品信息可以进行增加、删除、修改和查询；
5.	积分设置：管理员对会员消费后进行相应的积分设置；
6.	特权管理：管理员根据会员等级设置会员所拥有的特权；
7.	会员用户统计：实现管理员对已注册用户的统计，并可以进行修改和查询；
对会员用户所具有权限使用的模块
1.	登录管理：会员可以登录后查看自己的信息并可以修改自己账号的密码；
2.	消费记录：对会员用户的所有消费进行记录。会员用户可以查看和修改。
3.	积分管理:会员可以查看自己的积分；
4.	等级管理：会员可以查看自己的等级，以及升到下一等级所需要的积分，还
可以查看目前等级所拥有的特权；
5.	意见反馈：会员可以对已购买商品提出意见。





完成研究内容的技术路线或研究方法
技术路线：
拟采用mvc设计模式实现本系统的开发,在模型层使用Hibernate框架与数据库交互,使用JavaBean进行业务逻辑的封装,在视图层采用Jsp,JavaScript(JQuery)进行模板开发,在控制层采用Struts2进行控制流流转及视图与业务逻辑的分离,最后采用Spring进行框架的整合,通过依赖注入来降低模块间的耦合性并实现声明式事物处理。
开发语言：JSP,JAVA,Java Script
运行平台：Windows 7
本系统开发需求的软件是：Eclipse ,MySQL
研究方法：
采用软件工程的方法进行分阶段的开发,各阶段主要分为:需求分析阶段、概要设计阶段、详细设计阶段、编码阶段和测试阶段。
